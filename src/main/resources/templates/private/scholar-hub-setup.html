ScholarHub setup

Build your weekly schedule in one place

Start with the full working week, add subjects into any lesson slot, then drag cards to reorder them. This layout is simpler than the classic school matrix, scales better on mobile, and is easier to edit later.

Subject library

Existing subjects

Click a plus in the week, then choose a subject here. Cards can be dragged between slots after that.

Recommended backend flow

  • `GET /scholar-hub` checks for schedule and redirects to `/scholar-hub/schedule/setup` if missing.
  • The setup page edits the whole week in one payload, not day-by-day drafts.
  • On save, backend creates `Schedule -> Day -> DaySubject` in one transaction.
  • Reordering on the client only changes `lessonOrder` before submit.
